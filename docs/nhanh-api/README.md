# Tài liệu Nhanh.vn Open API v3.0

Thư mục này chứa các tài liệu API được thu thập từ Nhanh.vn Open API Documentation.

## Cấu trúc thư mục

```
docs/nhanh-api/
├── README.md                              # File này - Hướng dẫn sử dụng
├── NHANH_API_V3_DOCUMENTATION.md          # Tài liệu tổng quan về API v3.0
├── BILL_LIST_API.md                       # Tài liệu chi tiết API Danh sách hóa đơn
└── [Các file tài liệu API khác...]       # Các API endpoints khác sẽ được thêm vào đây
```

## Danh sách tài liệu

### Tài liệu tổng quan

- **[NHANH_API_V3_DOCUMENTATION.md](./NHANH_API_V3_DOCUMENTATION.md)**
  - Giới thiệu về NhanhAPI v3.0
  - Request Parameters, Paginator, Response Format
  - Error Codes và API Rate Limit
  - Danh sách tổng hợp các API Endpoints

### Tài liệu API chi tiết

- **[BILL_LIST_API.md](./BILL_LIST_API.md)**
  - API: Danh sách hóa đơn (`/v3/bill/list`)
  - Lấy danh sách phiếu xuất nhập kho: hóa đơn nhập nhà cung cấp, bán lẻ, bán sỉ, chuyển kho, kiểm kho
  - Cập nhật: 2024-12-04

## Cách sử dụng

1. **Đọc tài liệu tổng quan trước:** Bắt đầu với `NHANH_API_V3_DOCUMENTATION.md` để hiểu cấu trúc chung, cách xác thực, và các quy tắc chung.

2. **Tham khảo API cụ thể:** Mỗi API endpoint quan trọng sẽ có file tài liệu riêng với:
   - Mô tả chi tiết
   - Request/Response format
   - Ví dụ sử dụng
   - Các tham số và filters

3. **Cập nhật tài liệu:** Khi có API endpoint mới cần tài liệu chi tiết, hãy thu thập và tạo file mới trong thư mục này.
